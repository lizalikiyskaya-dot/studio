# Типографика сайта — единое правило (актуальная версия)

Финальная система — четыре уровня. Засечки остаются для заголовков и длинных текстовых полей (это читается как литературный текст), но мелкий интерфейс (кнопки, лейблы) — без засечек, и это НЕ Inter (слишком клишированный AI-шрифт, к тому же засечки на мелком кегле читаются плохо — два разных довода для одной правки).

## 1. Заголовки
(H1, названия разделов, масштхэд, названия карточек)
- Шрифт: **Old Standard TT**, serif (с засечками)
- `text-transform`: none, `letter-spacing`: normal

## 2. Длинные текстовые поля
(рукописи, ответы ученика, план книги, синопсис, любой объёмный текст для чтения/печати)
- Шрифт: **Old Standard TT**, serif (с засечками) — здесь засечки уместны, это читается как литературный текст, не как интерфейс
- `text-transform`: none, `letter-spacing`: normal

## 3. Интерфейс
(кнопки, лейблы полей форм, навигация, короткие подписи — всё мелкое и короткое, что не читается абзацами)
- Шрифт: **Golos Text**, sans-serif (без засечек) — ЗАМЕНЯЕТ Inter
- Почему не Inter: слишком предсказуемый/клишированный шрифт AI-сайтов, плюс хуже задумывался для очень мелкого кегля, чем Golos Text, который специально проектировался под читаемость на экране в интерфейсах
- `text-transform`: none, `letter-spacing`: normal

## 4. Мелкие технические метки
(статус-бейджи, номера разделов «01»/«02», % в таблицах, даты, лейблы-теги)
- Шрифт: **JetBrains Mono**, monospace
- Единственное место, где разрешён `text-transform: uppercase` и небольшой `letter-spacing` (0.03–0.05em)

## Как различить пункт 2 и пункт 3
Если это поле, куда печатают связный текст (предложениями/абзацами) — пункт 2, с засечками.
Если это слово/короткая фраза на кнопке, лейбле, в навигации — пункт 3, без засечек.

## Глобальное правило
`text-transform: uppercase` запрещён везде, КРОМЕ пункта 4.

## Что нужно сделать Claude Code
1. Подключить шрифт Golos Text (Google Fonts) вместо Inter
2. Применить Golos Text ко всем кнопкам, лейблам форм, навигации, коротким интерфейсным подписям
3. Оставить Old Standard TT для заголовков И для текстовых полей ответов/рукописей/плана книги (где печатают длинный текст)
4. Проверить весь сайт целиком: логин, регистрация, кабинет наставника (включая «УЧЕНИКИ»), все вкладки кабинета ученика — найти каждое место, где раньше стоял Inter, и заменить на Golos Text (если это интерфейс) или оставить Old Standard TT (если это длинное текстовое поле)
