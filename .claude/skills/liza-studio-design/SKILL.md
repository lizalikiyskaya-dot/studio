# Дизайн-система студии Лизы Ликийской

Источник истины — файл `platform_redesign_v11.html` (кликабельный макет, если приложен к проекту) и этот документ. Перед правкой глобального CSS/токенов — после изменения сверься, что ни одна страница не осталась со старыми значениями (grep по всему репозиторию).

ВАЖНО: эта версия дизайна полностью заменяет более ранние версии (газетную/serif-heavy и промежуточную "плавающая карточка"). Если в коде встретятся шрифты Old Standard TT, PT Serif, Golos Text, Manrope, Inter-как-замена — это пережитки старых версий, заменить по таблице ниже.

## 1. Шрифты — три уровня
| Уровень | Шрифт | Где |
|---|---|---|
| Заголовки | **Source Serif Pro** (serif), Bold | масштхэд, H1, названия разделов, названия карточек |
| Весь остальной текст | **Inter** (sans-serif) | подписи полей, кнопки, навигация, текст для чтения (рукописи, ответы ученика, план книги) |
| Мелкие метки | **JetBrains Mono** (monospace) | статус-бейджи, номера разделов, % в таблицах, даты |

`text-transform: uppercase` запрещён везде, КРОМЕ мелких меток.

## 2. Цвета интерфейса (точные значения, не выдумывать заново)
```
--bg-page (фон сайдбара):              #F8F4F1
--bg-card (фон основного контента):     #FFFFFF
--bg-surface-2 (KPI-плашки, hover):     #FAF8F4
--ink:                                  #1F201E
--ink-soft:                             #5C5B56
--ink-faint:                            #8E8B84
--border:                               #E7E4DC
--accent / бордовый (primary):          #873443
--accent-soft:                          #F0E1E4
--sage / зелёный:                       #4C684B
--sage-soft:                            #E2EADC
```
Активная плашка навигации, кнопка сворачивания сайдбара, линия под масштхэдом — все три используют #E8E1DC (нейтральный бежевый, НЕ бордовый, НЕ короткая яркая линия).
Плашка «Режим правок» и виджет календаря — белые (#FFFFFF), не --bg-surface-2.

## 3. Палитра для пользовательского выбора цвета (трекер книг, сюжетные линии, будущее выделение текста)
11 цветов, использовать именно эти значения:
```
Тёмные: #4C684B (зелёный), #1F201E (почти чёрный), #873443 (бордовый)
Светлые: #F0E1E4 (розовый), #E0CBE7 (лаванда), #E2E9EE (голубой), #E2EADC (светло-зелёный),
         #E8E1DC (бежевый), #F4DDBC (персиковый), #F8F4F1 (светло-кремовый), #FFFFFF (белый)
```
Эта палитра — для пользовательского выбора (раскраска ячеек, будущие заметки/выделение текста), НЕ для элементов самого интерфейса, кроме случаев из пункта 2.

## 4. Иконки
Набор **Lucide** (`lucide-react` — обычный npm-пакет для Next.js, импорт по имени, без файлов-картинок):
```
Задания → Bookmark · Материалы → BookOpen · Рукописи → FileText · Упражнения → Pencil
Принято → Check · В процессе → Clock · На доработке → RotateCcw · Сворачивание сайдбара → PanelLeft
```

## 5. Расположение
Сайдбар прижат к левому краю окна браузера, контент — к правому, без отступов вокруг, разделены тонкой линией `var(--border)`. НЕ заворачивать всё в «плавающую карточку» со скруглением/тенью/отступами по краям экрана — это отклонённый промежуточный вариант.

## 6. Масштхэд
«Студия художественной прозы» (Source Serif Pro bold) → «Лизы Ликийской» (Source Serif Pro обычное начертание, без курсива) → тонкая линия на всю ширину блока цвета #E8E1DC.

## 7. Компоненты
- Скругления: 14px на карточках, полное скругление (pill) на статус-бейджах и активных кнопках
- Тени мягкие и рассеянные, не жёсткие рамки
- Сайдбар: компактные строки иконка+текст (НЕ крупные залитые кнопки одна под одной), активный пункт — флэт нейтральный фон #E8E1DC, текст --ink (НЕ перекрашивается в акцентный цвет)
- Кнопка сворачивания сайдбара в иконочную полоску — видимый квадрат с иконкой PanelLeft рядом с масштхэдом
- KPI-плашки на дашборд-страницах: иконка (Lucide, не текстовый символ) в цветном круге + крупное число + мелкий лейбл
- Маячки непрочитанных правок — точка акцентного цвета с мягкой расходящейся пульсацией (анимация, не резкое мигание)
- Заметки — плавающая круглая кнопка (FAB) в правом нижнем углу с бейджем-счётчиком, по клику панель выезжает справа

## 8. Структура меню
**Верх (крупные пункты):** Задания и обратная связь, Материалы, Рукописи, Упражнения
**Мастерская книги:** План книги, О книге, Сеттинг (+ подвкладка «Фэнтези мир» с замочком открытия для конкретного ученика), Досье персонажей, Арки персонажей (5 арок: позитивная, разочарования, падения, порчи, плоская — три последних это разные виды "отрицательной" арки по K.M. Weiland), Акты и сюжетные линии
**Мастерская рассказов** (включается/выключается отдельно для конкретного ученика, дополняет книжную, не заменяет): 01 Циклы рассказов, 02 Отдельные рассказы

## 9. Перед тем как закончить любую UI-задачу
1. Проверь страницу логина, регистрации, кабинет наставника И кабинет ученика (все вкладки) — общий компонент встречается везде
2. Прогони grep по `uppercase` и по названиям шрифтов, которых не должно быть (Old Standard TT, PT Serif, Golos Text, Manrope) — если найдёшь, замени по таблице из пункта 1
3. Прогони grep по признакам старой «плавающей карточки» (закруглённый контейнер всего макета с тенью и отступами от края экрана) — если найдёшь, убери, макет должен быть во весь экран
4. Если правишь что-то, что выглядит как разовая правка одного экрана — спроси себя, не общий ли это компонент, который повлияет на другие страницы
